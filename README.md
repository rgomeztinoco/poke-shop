# POKE SHOP

Game where you capture pokemons and then you can shop them. Created with Ruby on Rails and React.js

---

How to run in local:

- Clone the repo
- run `rails db:create` y `rails db:migrate`
- run `rails server`
- open `localhost:3000` in your favorite browser
